\section{Abstract}
This project presents an interactive Data Structures and Graph Algorithms Visualizer, built using C++ and the Raylib graphics library. The main purpose of this application is to provide a simple and user-friendly visualization application that can help users understand data stored in basic data structures.

Key features include the visualization of Trees (AVL, Trie, Heap), Linked List, and Graphs (Minimum Spanning Tree, Shortest Path), alongside a state-management system that allows users to skip animations to the final result, and review completed operations step-by-step using backward and forward controls. Additionally, the software features a customizable UI environment with modifiable color themes and node styles.
