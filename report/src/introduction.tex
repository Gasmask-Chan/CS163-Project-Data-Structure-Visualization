\section{Introduction}
Data structures and algorithms form the backbone of computer science. However, these concepts are often taught using static whiteboard drawings or dense textual descriptions, which fail to capture the dynamic nature of memory manipulation and recursive operations. Recognizing this educational bottleneck, this project was motivated by the desire to create a visual, interactive sandbox. An application where users can freely input data and immediately observe the structural changes without dealing with console outputs or debugger tools. The core objective is to deliver a stable visualization tool that handles real-time user inputs flawlessly. It seeks to accurately implement and animate six core data structures: AVL Tree, Max Heap, Singly Linked List, Trie, Minimum Spanning Tree (Kruskal's/Disjoint Set Union), and Shortest Path (Dijkstra's). Furthermore, the project strives to achieve a professional-grade user experience through robust error handling, memory safety, and visual customization.
