A simple visualizer for data structures and basic graph algorithms.

This visualizer supports the following data structures:
- Singly Linked List.
- Heap.
- AVL.
- Trie.

and graph algorithms such as:
- Minimum Spanning Tree using Kruskal's algorithm.
- Shortest Path using Dijsktra's algorithm.

## Technology Stack
- Language: C++.
    - Compiler: g++ (Rev8, Built by MSYS2 project) 15.2.0.
    - Build system: Makefile.
- Graphic: Raylib 5.5.
